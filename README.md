# 📦 Distribuidora — Sistema de Gestión Integral

La **Distribuidora** es un sistema completo para la administración de productos, inventarios, ventas y logística.  
Diseñado para ofrecer un flujo de trabajo eficiente, seguro y escalable para empresas que manejan operaciones de distribución.

---

## 🚀 Características Principales

- **Gestión de productos:** categorías, variantes, fotos, unidades.  
- **Control de inventario:** entradas, salidas, ajustes, alertas de stock.  
- **Ventas y pedidos:** creación, seguimiento y control de flujo.  
- **Logística:** rutas, entregas y estado de repartidores.  
- **Dashboard analítico:** métricas de ventas, rotación e indicadores.  
- **Usuarios y roles:** permisos personalizados.  
- **Reportes:** exportación en Excel y PDF.  
- **Autenticación segura:** basada en API tokens.

---

## Sobre el Proyecto

Este sistema busca ofrecer una solución sólida para distribuidoras que requieran:

- Minimizar errores de inventario  
- Acelerar operaciones internas  
- Optimizar rutas y entregas  
- Mejorar el flujo de ventas  
- Centralizar la información del negocio  
- Obtener analíticas en tiempo real  

Desarrollado con una arquitectura moderna basada en **API REST**, permitiendo que frontend y backend estén desacoplados para mayor escalabilidad.
